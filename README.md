# My Queue (ref_qeueu) - Mobile App

**My Queue** is a digital queue management application designed for refugee health operations. 
This repository contains the **Mobile Application** which focuses entirely on the refugee experience and emergency ambulance services.

*(Note: The professional dashboards for Doctors, Pharmacy, and Lab technicians are housed in a separate Web Application project).*

## 📱 Mobile App Scope (This Project)
1. **Refugees (Patients)**
   - Request medical help by submitting symptoms and images.
   - Join hospital queues digitally based on location.
   - Track live queue status and estimated wait times.
2. **Ambulance Drivers**
   - Receive emergency requests.
   - Provide live location tracking to the patient.
   - Update estimated time of arrival (ETA) and dispatch status.

## 💻 Web App Scope (Separate Project)
- **Doctors**: View queues, conduct consultations, build prescriptions, and request lab tests.
- **Pharmacy**: View digital prescriptions and dispense medications to patients in the queue.
- **Lab Technicians**: Manage test requests and upload results.
